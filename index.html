Lesson 1. Hometask

Boolean

Преобразование примитивов

  • let a = 0 || 'string'; - возвратит true, т.к. логическое ИЛИ принимает значение true, когда хотя бы один из операндов true. Строка даст true, т.к. она не пустая
  • let a = 1 && 'string'; - возвратит true, т.к. логическое И принимает значение true, когда все операнді true. Строка даст true, т.к. она не пустая
  • let a = null || 25; - вернет true. Любое число, кроме 0 и -0 дает true.
  • let a = null && 25; - вернет false, т.к. null => false, 25 => true
  • let a = null || 0 || 35; - вернет true, т.к. null => false, 0 => false, но 35 => true.
  • let a = null && 0 && 35; - вернет false, т.к. null => false, 0 => false, 35 => true.

Что отобразится в консоли

  • 12 + 14 + '12'; - вернет строку "2612". Сначала посчитается сумма 12 + 14, затем попытается проссумировать число со строкой и произойдет неявное преобразование к строке, т.к. отсутствует унарный плюс возле строки "12".
  • 3 + 2 - '1'; - вернет число 4. Произойдет неявное преобразование из строки "1" в число 1. Преобразование числа в строку происходит только при наличии бинарного плюса.
  • '3' + 2 - 1; - вернет число 31. Сначала произойдет конкатенация строки "3" неявно преобразованного числа к строке бинарным плюсом "2". Затем строка "32", опять неявно, преобразуется к числу 32 при наличии математического оператора минус.
  • true + 2 - вернет число 3. true преобразуется к 1.
  • +'10' + 1; - вернет 11. Унарный + перед строкой "10" преобюразует ее к числу, а дальше простая математика
  • undefined + 2; - вернет NaN, т.к undefined преобразуется в NaN
  • null + 5; - вернет 5, т.к. null преобразуется в 0
  • true + undefined; - вернет NaN, т.к. true => 1, undefined => NaN
